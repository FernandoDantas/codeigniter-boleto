# CodeIgniter Boleto

Esse projeto possui as devidas adaptações no [Boleto PHP](https://github.com/CobreGratis/boletophp/) para que boletos possam ser gerados através do [CodeIgniter](https://github.com/bcit-ci/CodeIgniter/).
